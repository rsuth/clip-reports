OnCue Detailed Reports Generator

Many attorneys got used to using TrialDirector's "Detailed Reports" with barcodes for each clip. OnCue does not offer an easy way to create these reports with barcodes. Use this tool to create TrialDirector style Detail Reports that work with your OnCue Designation lists

This tool works completely offline - it runs locally on in your browser and no data leaves your computer.

To use, export an XML file of your designation list from OnCue. Then use the file selector below to load the file and create the report.

Once the report is generated, you can print this webpage for use in trial.
